# Google Icons Picker

A native macOS application that is easily accessible from the taskbar built using [Electron](https://www.electronjs.org/).

## Setup

```bash
# Install dependencies
$ npm install

# Run in development mode
$ npm start

# Build native .app
$ npm run make
```
